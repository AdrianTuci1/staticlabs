# Pixtooth
## AI-assisted dental charting for clinical workflows

### Research Context
Pixtooth is a dental charting system for modern practices, developed to reduce friction in clinical documentation while preserving the precision required in patient care. The platform combines structured odontogram workflows with machine learning support for auto-charting.

The project studies how AI can assist clinicians without hiding the reasoning path or weakening professional review.

### Clinical Workflow
Pixtooth supports chairside charting, treatment planning, and case documentation in a unified interface. Machine learning models assist with visual interpretation and proposed chart entries, while the clinician remains responsible for validation and final record quality.

The objective is to shorten repetitive documentation loops and make dental records easier to maintain across appointments, teams, and treatment stages.

### Machine Learning Scope
The auto-charting layer is designed to identify clinically relevant visual patterns and prepare draft observations for review. Static Labs' research focus is on confidence-aware assistance, traceable corrections, and model behavior that adapts to the realities of dental practice rather than forcing clinicians into rigid software flows.

### Research Status
Pixtooth extends the lab's broader work on human-machine collaboration into clinical environments, where speed, trust, and auditability must be designed together.
